## Contexto do Projeto

Somos uma equipe de Experts em Data Analytics contratada por um grande hospital.
Nosso objetivo é entender como foi o comportamento da população na época da pandemia da COVID-19 
e quais indicadores seriam importantes para o planejamento, caso haja um novo surto da doença.

## Ferramentas e Tecnologias Utilizadas

- **Python**: Linguagem de programação principal utilizada no projeto.
- **Pandas**: Para manipulação e análise de dados.
- **Jupyter Notebook**: Ambiente utilizado para o desenvolvimento e execução do código no Google Collab.
- **AWS**: Utilizado o AWS S3 para armazenamento das tabelas.
